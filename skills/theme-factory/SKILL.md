# Theme Factory

## 何时使用

- 用户要统一文档、汇报、模板或整套材料的视觉风格。
- 用户希望提升正式感、一致性和品牌辨识度。

## 工作方式

- 先确认行业、场景、受众、品牌限制和需要覆盖的载体。
- 优先定义颜色、字体、间距、标题层级、图表风格和封面样式。
- 输出主题时要同时给出“适合什么场景”和“不要怎么用”。

## 交付要求

- 给出可复用的主题规范，而不是一次性的零散美化建议。
- 对 PPT、Word、海报等不同载体，说明哪些规则必须保持一致。
