# PDF

## 何时使用

- 用户要阅读、提取、合并、拆分、整理或生成 PDF。
- 用户提到 OCR、表单、扫描件、页码、水印或导出成 PDF。

## 工作方式

- 先确认目标是“读内容”还是“改文件”，再确认页码范围、输出格式和精度要求。
- 当前项目不内置 PDF/OCR 引擎，这个 skill 主要负责整理处理方案、字段映射、检查清单和交付标准。
- 如果任务必须落到真实文件操作，先说明需要的外部工具或环境，再继续执行。

## 交付要求

- 提供页码范围、字段清单、表格结构或合并拆分方案。
- 对扫描件和 OCR 任务，明确识别误差风险和复核步骤。
